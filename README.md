# Web Chess GUI

This repository contains a simple React + TypeScript application created with [Vite](https://vitejs.dev/). The app renders a chess board using **react-chessboard** and manages game state with **chess.js**.

## Getting Started

The project files live inside the `web-chess` directory.

```bash
cd web-chess
npm install
npm run dev
```

This starts the development server on `http://localhost:5173` by default.

To create a production build run:

```bash
npm run build
```
