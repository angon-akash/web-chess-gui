# Web Chess GUI

A simple experiment building a web-based chess interface.

## Project Goals

- Provide an interactive chess board in the browser
- Use chess.js for validating moves and tracking game state
- Offer a basis for experimenting with additional features such as playing against an engine or storing games

## Stack

- **React** for the user interface
- **chess.js** for chess logic
- **react-chessboard** for rendering the board
- **Node.js** for running the development environment

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
   This should open the app at `http://localhost:3000`.
3. Build for production with:
   ```bash
   npm run build
   ```
